# Full Stack Open - Part 1: Introduction to React

This repository contains my solutions for **Part 1** of the Full Stack Open course from the University of Helsinki.

The focus of these exercises is to master the fundamentals of **React**, including component-based architecture, state management, and event handling.

### 📚 Learning Objectives:
- Understanding the structure of a React project using **Vite**.
- Managing application state with the `useState` hook.
- Handling events and passing functions as props to child components.
- Implementing **conditional rendering** based on application state.
- Practicing **immutability** when updating arrays and objects in state.

### 🛠️ Exercises:

- **[Course Info](./courseinfo)**  
  *A simple application that displays course details. It demonstrates how to pass data from a parent component to multiple child components using props.*

- **[Unicafe](./unicafe)**  
  *A feedback collection app that tracks 'good', 'neutral', and 'bad' ratings, displaying real-time statistics (average, positive percentage) in a structured HTML table.*

- **[Anecdotes](./anecdotes)**  
  *A random anecdote generator that allows users to vote for their favorite programming quotes and dynamically identifies the anecdote with the highest number of votes.*
